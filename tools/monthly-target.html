How it is calculated

  • Daily Target = Monthly Target ÷ Working Days
  • Weekly Target = Monthly Target ÷ Weeks in Month (or Daily Target × 7)
  • Orders/Day = Daily Target ÷ AOV

Tip: If your daily target feels too high, increase AOV (bundles), improve conversion, or add one more sales channel.
